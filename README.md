# Coffee-Sales-Dashboard-Power-BI-Project
 Coffee Sales Performance Dashboard – Power BI Project

📌 Project Overview

The Coffee Sales Performance Dashboard is an interactive Power BI report designed to analyze coffee shop sales data and provide actionable business insights. The dashboard helps stakeholders monitor sales performance, identify top-selling products, understand customer purchasing patterns, and evaluate store-level performance for better decision-making.




🎯 Problem Statement

Coffee shop businesses generate large volumes of sales data daily, making it difficult to manually track performance and identify trends. Management requires a centralized dashboard to:

Monitor overall sales performance.

Track product-wise and category-wise sales.

Identify peak sales periods during the day.

Analyze customer payment preferences.

Compare store performance across locations.

Support data-driven business decisions.


The objective of this project is to transform raw sales data into meaningful visual insights through an interactive Power BI dashboard.

📌 Business Insights

✔️ Coffee products contribute the highest revenue.
